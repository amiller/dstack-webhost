## Conventions

- One file per RFC: `NNNN-short-name.md`
- Status: Draft → Accepted → Implemented (or Withdrawn)
- RFCs are discussion documents, not commitments. Write them when you have
  an idea worth thinking through, merge them when the direction is clear.
